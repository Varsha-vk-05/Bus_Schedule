# 📱 BusTime – Bus Schedule & Live Tracking App

## 🚀 Overview
**BusTime** is an Android application developed during my internship at MindMatrix, designed to help users easily access bus schedules and track buses in real time.

The app provides a smooth and user-friendly interface to view bus timings, explore routes, and track buses live using Google Maps integration.

---

## ✨ Features

- 🚌 Bus Schedule Listing  
  View all available bus stops along with arrival timings.

- 🔍 Search & Filter  
  Search buses by stop name or bus number.

- ⭐ Favorites  
  Save frequently used stops for quick access.

- 🗺️ Live Bus Tracking  
  Track buses in real-time using Google Maps.

- 📍 Route Visualization  
  View routes between stops with markers and paths.

- 🎨 Modern UI Design  
  Clean and intuitive interface for better user experience.

---

## 🛠️ Tech Stack

- Language: Kotlin  
- Framework: Android SDK  
- UI: Jetpack Compose  
- Maps: Google Maps SDK  
- Architecture: MVVM (basic implementation)
